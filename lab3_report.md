# Лабораторная работа №3

## Фильтрация изображений и морфологические операции

### Вариант

Для варианта `16` по таблице из задания требуется выполнить **морфологическое расширение (дилатацию)** для бинарного изображения со структурирующим элементом **диск `3 x 3`**.

Используемый структурирующий элемент:

```text
0 1 0
1 1 1
0 1 0
```

Центр структурирующего элемента находится в середине маски.

### Цель работы

1. Взять те же изображения рукописей, которые использовались во второй лабораторной работе.
2. Использовать бинарные результаты из лабораторной работы №2 как вход для морфологической обработки.
3. Применить морфологическое расширение в соответствии с вариантом `16`.
4. Построить разностное изображение `XOR` между входным и выходным бинарным изображением.

### Исходные данные

В работе использованы те же два изображения рукописей, что и во второй лабораторной работе:

- `manuscript_01`
- `manuscript_02`

Так как вариант `16` относится к морфологической обработке бинарных изображений, на вход операции подаются не исходные цветные фотографии напрямую, а результаты бинаризации из лабораторной работы №2.

В качестве бинарного входа использованы изображения, полученные методом **Брэдли и Рота** с окном `15 x 15`, что соответствует варианту `16` второй лабораторной работы.

### Теоретические сведения

#### 1. Морфологическое расширение

Морфологическое расширение используется для увеличения областей переднего плана на бинарном изображении. В данной работе передним планом считаются **черные пиксели текста и темных объектов**, а белый цвет считается фоном.

Идея операции такая: для каждого пикселя рассматривается его локальная окрестность `3 x 3`. Если хотя бы один черный пиксель из входного изображения попадает под единичные элементы структурирующего элемента, то соответствующий пиксель результата также становится черным.

Для варианта `16` применяется маска:

```text
0 1 0
1 1 1
0 1 0
```

Это означает, что расширение идет по четырем основным направлениям и по центру. По сути, темные области слегка утолщаются, мелкие разрывы могут частично закрываться, а отдельные тонкие штрихи становятся более выраженными.

#### 2. Разностное изображение

По условию задания дополнительно строится разностное изображение. Для бинарного случая удобно использовать операцию `XOR`:

- белый пиксель на разностном изображении означает, что значение изменилось
- черный пиксель означает, что пиксель остался без изменений

Такое изображение позволяет наглядно увидеть, где именно дилатация добавила новые черные области.

### Реализация

В работе выполнены следующие шаги:

1. Загрузка исходных цветных изображений рукописей.
2. Загрузка бинарных изображений из лабораторной работы №2.
3. Интерпретация черного цвета как объекта морфологической обработки.
4. Применение дилатации со структурирующим элементом `3 x 3` типа «диск».
5. Построение `XOR`-разности между входным и выходным изображением.
6. Сохранение результатов для каждого изображения.

Для каждого файла сохранены:

- `00_source.png` — исходное цветное изображение
- `01_binary_input.png` — бинарный вход из лабораторной работы №2
- `02_variant16_dilation.png` — результат дилатации варианта `16`
- `03_xor_difference.png` — `XOR`-разность
- `04_panel.png` — сводная панель для наглядного сравнения

### Вывод

В лабораторной работе №3 для двух изображений рукописей была выполнена морфологическая операция, соответствующая варианту `16`: **дилатация** со структурирующим элементом «диск» размера `3 x 3`.

Результаты показывают, что такая операция:

- увеличивает толщину темных штрихов
- частично закрывает мелкие разрывы
- усиливает контуры текста
- одновременно может делать буквы более слитыми при плотном расположении символов

Дополнительно для каждого изображения построена `XOR`-разность, по которой удобно анализировать области изменений. В целом операция соответствует своему назначению и подходит для усиления темных объектов на бинарных изображениях.
