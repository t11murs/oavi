# Лабораторная работа №2
## Обесцвечивание и бинаризация растровых изображений

### Вариант

По таблице из задания вариант `16` соответствует методу **адаптивной бинаризации Брэдли и Рота** с размером окна **15 × 15**.

### Цель работы

Целью лабораторной работы является приведение полноцветных изображений к полутоновому виду и последующая бинаризация полутоновых изображений методом, соответствующим заданному варианту, без использования библиотечных функций обесцвечивания и пороговой обработки.

### Исходные данные

В работе использованы два исходных полноцветных изображения рукописей в формате PNG:

- `12.png` — рукописная страница с неравномерным освещением и слабым текстом
- `08.png` — страница старопечатного текста с выраженными символами и деформациями листа

Размеры изображений:

| Файл | Размер |
|:--|--:|
| `12.png` | 1320 × 2048 |
| `08.png` | 1314 × 2048 |

### Теоретические сведения

#### 1. Приведение полноцветного изображения к полутоновому

Для получения полутонового изображения яркость каждого пикселя вычислялась как взвешенное среднее каналов `R`, `G`, `B` по формуле:

`Y = 0.299R + 0.587G + 0.114B`

Такой способ позволяет получить одноканальное изображение яркости без использования готовых библиотечных функций обесцвечивания.

#### 2. Адаптивная бинаризация Брэдли и Рота

Для каждого пикселя рассматривается локальное окно `15 × 15`, внутри которого вычисляется средняя яркость. После этого текущий пиксель сравнивается не с глобальным порогом, а с локальным порогом, зависящим от среднего значения по окну.

Использованное решающее правило:

- если яркость пикселя меньше локального среднего, уменьшенного на долю `t`, то пиксель считается чёрным
- иначе пиксель считается белым

В работе использован параметр:

- `w = 15`
- `t = 0.15`

#### 3. Ускорение вычислений

Чтобы быстро находить сумму яркостей в окрестности каждого пикселя, использовано интегральное изображение. Это позволяет вычислять сумму по окну за постоянное время и эффективно обрабатывать изображения большого размера.

### Реализация

В рамках лабораторной работы были выполнены следующие этапы:

1. Загрузка исходного цветного изображения.
2. Ручное вычисление полутонового изображения по формуле `BT.601`.
3. Построение интегрального изображения яркости.
4. Бинаризация методом Брэдли и Рота с окном `15 × 15`.
5. Сохранение всех промежуточных и итоговых результатов.

Для каждого изображения сохранены:

- `00_source.png` — исходное изображение
- `01_grayscale.bmp` — полутоновое изображение в формате BMP
- `01_grayscale_preview.png` — PNG-превью полутонового изображения для отчёта
- `02_bradley_roth_w15.png` — результат бинаризации методом Брэдли и Рота
- `03_panel.png` — сводное изображение для сравнения этапов обработки

### Вывод

В ходе лабораторной работы были реализованы два основных этапа обработки растровых изображений:

1. Полноцветные изображения приведены к полутоновому виду по формуле взвешенного усреднения каналов `R`, `G`, `B`.
2. Полутоновые изображения бинаризованы методом **Брэдли и Рота**, соответствующим **варианту 16**, с использованием окна `15 × 15`.

По результатам обработки можно сделать вывод, что адаптивная бинаризация хорошо подходит для старых рукописных и старопечатных документов, поскольку учитывает локальные изменения яркости. Это позволяет лучше выделять текст на неоднородном фоне по сравнению с глобальным порогом, хотя при наличии сильных загрязнений и повреждений полностью избавиться от шумов не удаётся.
